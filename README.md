# Colony Survival 2D

A mobile-friendly 2D colony survival game built with HTML5 Canvas and JavaScript. Playable directly in web browsers and optimized for GitHub Pages deployment.

## 🎮 Game Features

### Core Gameplay
- **Building System**: Place different types of buildings on a grid
- **Colonist Management**: AI-controlled colonists that work and survive
- **Enemy Waves**: Defend against increasingly difficult enemy attacks
- **Resource Management**: Collect and manage food, wood, and stone
- **Mobile Optimized**: Touch controls and responsive design

### Buildings Available
- 🏠 **House** - Increases population capacity
- 🌾 **Farm** - Produces food for colonists
- 🪓 **Lumberyard** - Generates wood resources
- ⛏️ **Quarry** - Mines stone resources
- 🗼 **Tower** - Defends against enemies

### Game Systems
- **Wave-based Combat**: Enemies spawn in waves every 60+ seconds
- **Resource Production**: Buildings automatically generate resources
- **Colonist AI**: Colonists flee from enemies, collect resources, and maintain buildings
- **Mobile Controls**: Touch-friendly interface with gesture support

## 🎯 How to Play

### Desktop Controls
- **Click** building buttons to select a building type
- **Click** on the game canvas to place buildings
- **Mouse hover** shows building preview
- Use the control buttons to pause/speed up the game

### Mobile Controls
- **Tap** building buttons to select
- **Double-tap** on canvas to place buildings
- **Single tap and hold** shows building preview
- Use floating control buttons for game management

### Survival Tips
1. **Build Houses** early to increase population
2. **Place Farms** to keep colonists fed
3. **Construct Towers** near important buildings for defense
4. **Manage Resources** carefully - each building has costs
5. **Watch the Wave Timer** - prepare defenses before attacks

## 🚀 Deployment

### GitHub Pages Setup
1. Enable GitHub Pages in repository settings
2. Set source to main branch
3. Game will be available at: `https://yourusername.github.io/repository-name`

### Local Development
1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required - runs directly in browser
